# Keyboard Shortcuts

All keyboard shortcuts for the JetBeans admin panel.

**Legend:**
- `⌘` = Command (Mac) / Ctrl (Windows/Linux)
- `⌥` = Option (Mac) / Alt (Windows/Linux)
- `⇧` = Shift
- `↵` = Enter
- `⎋` = Escape

---

## Navigation

| Shortcut | Action | Notes |
|----------|--------|-------|
| `⌘ K` | Open search / command palette | Search anything |
| `⌘ B` | Toggle sidebar | Show/hide nav |
| `⌘ ⇧ H` | Go to Dashboard | Home |
| `⌘ ⇧ O` | Go to Orders | |
| `⌘ ⇧ P` | Go to Products | |
| `⌘ ⇧ C` | Go to Customers | |
| `⌘ ⇧ A` | Go to Analytics | |
| `⌘ ⇧ M` | Go to Messages | |
| `⌘ ,` | Go to Settings | Standard macOS convention |
| `⌘ [` | Go back | Browser back |
| `⌘ ]` | Go forward | Browser forward |

---

## Actions

| Shortcut | Action | Notes |
|----------|--------|-------|
| `⌘ ⇧ N` | New Product | Opens product create |
| `⌥ ⇧ O` | New Order | Opens order create |
| `⌘ S` | Save | Triggers save on current form |
| `⎋` (Escape) | Close / Cancel | Close modal, cancel action |

---

## View

| Shortcut | Action | Notes |
|----------|--------|-------|
| `⌘ ⇧ L` | Toggle theme | Light ↔ Dark mode |

## Feedback

Does this feel right? Consider:

1. **Conflicts** — Do any shortcuts conflict with browser/OS defaults?
2. **Discoverability** — Can users find these without docs?
3. **Consistency** — Do similar actions have similar shortcuts?
4. **Memorability** — Are the mnemonics logical? (P=Products, O=Orders, etc.)

### Suggested Changes

- [ ] `⌘ ⇧ H` for Home feels weird — maybe just `⌘ 1` for first nav item?
- [ ] Numbered shortcuts: `⌘ 1` Dashboard, `⌘ 2` Orders, `⌘ 3` Products...
- [ ] `G` prefix for "Go to" sequences (like GitHub): `G O` = Orders, `G P` = Products
- [ ] Table navigation: `J/K` up/down, `Enter` to open, `X` to select
